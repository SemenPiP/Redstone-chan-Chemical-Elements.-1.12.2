This mod is a 1.12.2 port of Hongshijiang's Chemical Elements, a mod originally developed for higher Minecraft versions.
This mod adds 118 chemical elements and their derived items to the game.
There are no natural acquisition methods for these items, only basic crafting recipes are provided. It is designed specifically for modpack authors to perform custom modifications and tweaks.
Changes made in this port are as follows:
Only three block types are retained: Storage Blocks, Raw Blocks, and Ores with vanilla Minecraft textures. All item materials remain unchanged.
The vanilla Minecraft font does not support the Chinese characters for the following elements: Bohrium (Bh), Darmstadtium (Ds), Dubnium (Db), Flerovium (Fl), Hassium (Hs), and Roentgenium (Rg), which will cause display anomalies. This is an inherent defect of the original font. Custom new glyphs have been created for these characters in this port to ensure their normal usability.
