# redstone_chemical_elements

Forge 1.12.2 mod project for `Redstone-chan Chemical Elements`.

This repository contains the converted 1.12.2 source project, resources, Gradle wrapper, and helper scripts used to rebuild assets/fonts.

## Environment

- Minecraft: `1.12.2`
- Forge/MCP mapping: `stable_39`
- Build plugin: `RetroFuturaGradle`
- Java toolchain target: `Java 8`

## Build

On Windows:

```bat
gradlew.bat clean build --no-daemon
```

Build outputs are generated under:

```text
build\libs
```
